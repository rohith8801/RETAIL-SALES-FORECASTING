# 🛒 BigMart Sales Prediction

A Flask-based Machine Learning web application that predicts product sales for BigMart stores using product and outlet characteristics. The application integrates a trained regression model with a responsive web interface to provide real-time sales predictions.

---

## 📖 Overview

Accurate sales forecasting helps retailers optimize inventory management, pricing strategies, and business planning. This project applies Machine Learning techniques to estimate product sales based on historical product and outlet information.

Users can enter product details through an intuitive web interface and receive instant sales predictions generated by the trained model.

---

## ✨ Features

* Predicts BigMart product sales in real time
* Interactive and responsive Flask web interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/BIGMART.git
```

### Navigate to the project

```bash
cd BIGMART/app
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 📥 Input Parameters

The application predicts sales using the following product and outlet attributes:

* Item Weight
* Item Fat Content
* Item Visibility
* Item Type
* Item MRP
* Outlet Identifier
* Outlet Establishment Year
* Outlet Size
* Outlet Location Type
* Outlet Type

## 📤 Output

The application predicts the **estimated sales** for the selected BigMart product.
